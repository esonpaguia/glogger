# GLogger Check In

  Logs you in GLogger automatically.

## How to execute

  Syntax:
  ```
  java -jar glogger-checkin-0.0.1-SNAPSHOT-jar-with-dependencies.jar *[CONFIG_FILE_PATH]*
  ```

  Example: 
  ```
  java -jar glogger-checkin-0.0.1-SNAPSHOT-jar-with-dependencies.jar config.properties
  ```
